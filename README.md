# CourseWise (CourseLLM)

CourseWise is a course assistant web app built with **Next.js**, **Firebase** (Auth, Firestore, Storage, Emulators), and a **Python DSPy backend** for LLM-based features.

## How to run locally

See **`PRD.md`** in the project root for full local setup and run instructions  
(Firebase emulators, Python backend, Next.js dev server).

## Project structure (high level)

- `src/app` – Next.js app routes (student/teacher dashboards, auth, etc.)
- `src/ai` – Python DSPy backend and flows
- `docs/` – feature PRDs and design docs
